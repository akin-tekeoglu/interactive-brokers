# About

Some utility tools for ibkr tws api

# Requirements

- Python 3.9
- pipenv
- IBKR TWS API (https://interactivebrokers.github.io)


# Installation

- Move ibkr api under the `IBJts` directory
- `pipenv install`


# Spoof detector

# Fin wiz fetcher
